# 📚 Estructura de Datos II

Repositorio destinado al almacenamiento de las prácticas, tareas y proyectos desarrollados durante la asignatura **Estructura de Datos II**, implementados en **Python 3**.

## 🎯 Objetivo

Aplicar los conceptos fundamentales de estructuras de datos mediante el desarrollo de programas que permitan comprender su funcionamiento, implementación y aplicación en la resolución de problemas.

## 🛠️ Tecnologías utilizadas

- 🐍 Python 3
- Visual Studio Code (o cualquier editor compatible)
- Git & GitHub

## 📂 Estructura del repositorio

```text
Estructura/
│
├── Tarea_01/
├── Tarea_02/
├── Tarea_03/
├── Practicas/
├── Proyectos/
└── README.md
````

Cada carpeta contiene el código fuente, archivos de apoyo y, cuando sea necesario, la documentación correspondiente.

## 🚀 Cómo ejecutar los programas

1. Clona el repositorio:

```bash
git clone https://github.com/MACS1024/Estructura.git
```

2. Accede al directorio:

```bash
cd Estructura
```

3. Ejecuta el archivo deseado:

```bash
python nombre_del_archivo.py
```

> En algunos sistemas puede ser necesario utilizar:

```bash
python3 nombre_del_archivo.py
```
